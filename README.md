Let’s Go Super Saiyan Today! — Workout Tracker

A gym-focused web app inspired by Dragon Ball Z, allowing users to track lifts, warm-ups, workout duration, and personal progress.

Features: Log exercises, sets, reps, and weights. Track warm-ups and workout duration. Motivational “Super Saiyan” themed UI. Local storage support to save user data. Mobile-friendly design for gym use

Tech Stack: HTML, CSS, JavaScript
